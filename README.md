# Devoir intermédiaire : Exploitation de données électorales avec Python

Ce dépôt contient notre rendu pour le devoir maison intermédiare du cours **Python pour la Data Science**. 

L'objectif de ce projet est d'explorer, nettoyer et analyser les données électorales du premier tour de l'élection présidentielle française de 2022. Le projet inclut le calcul des scores nationaux et départementaux, l'analyse de la surreprésentation des candidats, et la cartographie des résultats.

## Structure du dépôt

* `notebook_elections_2022.ipynb` : Le notebook Jupyter principal contenant l'intégralité de l'analyse, les commentaires, ainsi que la génération des tableaux et figures.
* `README.md` : Ce fichier de présentation.
* `requirements.txt`: la liste des librairies nécessaires au bon lancement du notebook

## Prérequis et Installation

Pour garantir la reproductibilité totale du code, assurez-vous d'avoir installé les bibliothèques nécessaire en lançant la première cellule du notebook.
